setActiveTab

setActiveTab: function(tab)

Sets the currently active tab.  The idea of a focused tab is useful for keyboard navigation and returning to the last zoomed-in tab.  Hiting return/esc brings you to the focused tab, and using the arrow keys lets you navigate between open tabs.

Parameters

  • Takes a TabItem
