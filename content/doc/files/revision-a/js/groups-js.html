Group

A single group in the tab candy window.  Descended from Item.  Note that it implements the Subscribable interface.

Functions

add

add: function(a,
dropPos,
options)

Adds an item to the group.  Parameters:

aThe item to add.  Can be an Item, a DOM element or a jQuery object.  The latter two must refer to the container of an Item.
dropPosAn object with left and top properties referring to the location dropped at.  Optional.
optionsAn object with optional settings for this call.  Currently the only one is dontArrange.

remove

remove: function(a,
options)

Removes an item from the group.  Parameters:

aThe item to remove.  Can be an Item, a DOM element or a jQuery object.  The latter two must refer to the container of an Item.
optionsAn object with optional settings for this call.  Currently the only one is dontArrange.

reorderBasedOnTabOrder

reorderBasedOnTabOrder: function()

Reorderes the tabs in a group based on the arrangment of the tabs shown in the tab bar.  It doesn’t it by sorting the children of the group by the positions of their respective tabs in the tab bar.

getChild

getChild: function(index)

Returns the nth child tab or null if index is out of range.

Parameters index - the index of the child tab to return, use negative numbers to index from the end (-1 is the last child)

onNextNewTab

onNextNewTab: function(callback)

Sets up a one-time handler that gets called the next time a tab is added to the group.

Parameters callback - the one-time callback that is fired when the next time a tab is added to a group; it gets passed the new tab
