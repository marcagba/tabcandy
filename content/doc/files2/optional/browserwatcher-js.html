WindowWatcher

BrowserWatcher

When this object is instantiated, the given onLoad() is called for all browser windows, and subsequently for all newly-opened browser windows.  When a browser window closes, onUnload() is called. onUnload() is also called once for each browser window when the extension is unloaded.
